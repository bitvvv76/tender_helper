# Tender Helper MVP

Tender Helper — это MVP тендерного помощника для бизнеса.

Проект создан как демонстрационный прототип, который показывает, как можно автоматизировать первичный поиск и разбор тендеров.

## Что умеет бот

- принимать текстовый запрос по тендерам;
- определять категорию запроса;
- определять регион;
- определять бюджет;
- сохранять запросы пользователя в SQLite;
- показывать историю запросов;
- подбирать похожие демо-тендеры;
- объяснять, какие тендеры могут подойти;
- показывать инструкцию по команде `помощь`.
